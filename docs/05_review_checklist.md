# 05 レビューチェックリスト

このチェックリストは、Codex 生成物の自己点検と人のレビューの両方で使うための最終確認表です。マージ前に、構成・整合・スコープの抜け漏れがないかを短時間で確認することを目的にしています。

## 構造
- [ ] 変更対象は指示されたファイルに限定されている
- [ ] 見出し構造が読みやすく、目的が明確
- [ ] README と docs の参照関係が成立している

## 命名と整合
- [ ] ファイル名・番号順・見出し名が既存ルールと一致
- [ ] 人 / ChatGPT / Codex の役割記述に矛盾がない
- [ ] 用語（幹スレッド / 枝スレッド / プルリク）が統一されている

## スコープ管理
- [ ] 要件外の新規ファイルや仕組みを追加していない
- [ ] 実装詳細ではなく運用ガイドに集中している
- [ ] 変更が小さく、レビュー可能な粒度である

## テンプレート有用性
- [ ] テンプレートがそのまま再利用できる
- [ ] 入力すべき項目が不足なく、過剰でもない
- [ ] 記入例が短く実用的

## マージ準備
- [ ] プルリク説明に目的・差分・確認結果が記載されている
- [ ] 未解決事項とフォローアップが明示されている
- [ ] 幹スレッドへ返す報告内容が整理されている
- [ ] 必要に応じて、マージ後に ChatGPT プロジェクト情報源へ反映する手順が整理されている
